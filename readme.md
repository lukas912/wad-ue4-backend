#WAD UE4 BACKEND

| Method | Url        | Description        |
|--------|------------|--------------------|
| GET    | /tasks     | Returns all tasks  |
| GET    | /task/{id} | Returns task by id |
| POST   | /task      | Creates new Task   |
| PUT    | /task      | Edit a task        |
| DELETE | /task/{id} | Deletes a task     |
